# 免责声明

本框架不是价值评判工具，
也不是“作品好坏排行榜”，
而是一个回答结构问题的分析系统

这个框架使用了结构主义叙事学进行编写，没有相关经验理解起来会有些困难
事实上，这个框架更加适合创作者去排除自己的问题

**额外补充一点，事实上一个故事的好与坏取决于故事本身以及作者讲故事的能力，一个决定上限，一个决定下限，这个框架更多是在评价故事本身，而许多看着觉得很甜的番事实上是作者讲故事的能力强，和故事本身无关**

这是我自己在写小说的时候写出来的框架，当时我为了写小说，看了许多恋爱漫画作为参考，发现很多漫画看着有意思，看完了之后却感觉什么都没写，为了找到这个原因，我写了这个框架出来

**目前来说，这个框架最大的两个问题就是**
评判过于主观，虽然我加了很多条件以及解释，但是还是非常容易变成主观判断

对于新手来说过于不友好，因为我在写的时候更多考虑准确性，而不是理解难易，等我什么时候闲下来应该会写一个简化版的



## 框架的适用范围

**仅适用于：**

- 以恋爱为主要叙事主线的作品
- 或虽非纯恋爱，但存在**完整、持续推进的恋爱关系**的作品

**不适用于：**

- 单集型恋爱段子
- 只有告白结果、没有关系过程的作品

# 一、关系是如何被“绑定”的

本分类是一种理想模型，在实际作品中通常以多种绑定方式混合出现

这里借用了马克斯韦伯的政治合法性模型中的名词，原因在于我的灵感来自于他的框架，所以我会保留原先的名词，但是意思完全不同

## 关系绑定的定义

**关系绑定 =** 在角色尚未互相选择之前，是什么力量使他们**必须持续发生关系**

本部分只回答一个问题：
这段关系**最初**是靠什么力量**强行成立**的

## 1. 记忆绑定型（传统型）

**核心问题：** 如果今天断绝关系，会损失什么？

**绑定来源：**
* 共同过去
* 共享记忆
* 不可复制的情感历史

两人不一定是喜欢对方，而是放不下对方，典型表现为青梅竹马、旧识重逢、曾经一起经历过某些关键事件

这是成本最低、观众接受度最高的绑定方式


## 2. 吸引绑定型（魅力型）

**核心问题：** 我明知道不理你更安全，为什么还是会靠近你？

**绑定来源：**
* 情绪吸引
* 审美吸引
* 才华 / 气质 / 性格张力

非常适合故事的开始，因为不需要任何其他东西，只要一个强设定就可以开始讲故事，但是非常依赖后续的冲突，一旦张力不足，关系就会失控



## 3. 结构绑定型（法理型）

**核心问题：** 即便我们不想见面，也逃不开彼此吗？

**绑定来源：**
* 规则
* 身份
* 环境
* 责任

我将结构绑定型分为两个子类
### 外部结构（外在法理型）
* 同住
* 同部门
* 上下级
* 家庭关系

### 内部结构（内在法理型）
* 道德责任
* 自我承诺
* 无法回避的选择后果

法理型绑定稳定、强制、难以切断，但如果不去主动打破这种关联，关系容易停留在“被迫共存”



# 二、冲突再生产（关系得以存在的必要条件）

本部分不讨论“有没有冲突”，而讨论三个更具体的问题：
1. 冲突从哪里来？
2. 冲突靠什么反复发生？
3. 冲突为什么结束不了？

## 冲突从哪里来？

### 外在障碍型
冲突来源于角色之外的因素：
* 身份
* 阶级
* 距离
* 疾病
* 家庭 / 社会压力

这是最安全、最传统的冲突来源


### 内在缺陷型
冲突来源于角色自身：
* 性格缺陷
* 价值观冲突
* 情感回避
* 自我否定

大量人物驱动型作品都依赖此类冲突



### 关系定义型
冲突来源于关系本身即是矛盾结构：
* 博弈
* 身份对立
* 权力不对等
* 角色互为工具

这类关系风险
* 冲突一旦消失，关系即失去定义
* 天生具有“告白即死”“转型困难”问题

（典型例子：《NANA》）



## 冲突靠什么反复发生？

### 博弈型（这里的博弈型不是指来源，而是指冲突的表现形式）

如果角色直接说实话，剧情立刻结束，那么这就是博弈型表达

表现为：
* 攻防转换
* 误判
* 反转
* 心理优势更替

### 日常扰动型

通过重复但可变的日常场景制造
* 上学
* 放学
* 吃饭
* 回家
* 小情绪

代表作：《高木》《月色真美》



### 任务 / 事件驱动型

角色因为某个目标或事件被暂时绑定：

* 解谜
* 合作
* 完成任务

**注意：** 单独说明一下，这种任务驱动本身并不代表感情，之所以有这个任务，目的只是让角色在完成任务的途中展现出性格、价值观、能力



## 冲突为什么结束不了？

### 不可撤回状态（情绪余震型）

事件型冲突的结构是：
1. 问题出现
2. 问题解决
3. 事件结束

而不可撤回状态关心的是：事情已经结束，关系还能回到原样吗？

一旦选择造成关系的永久性变化，即使外在冲突消失，情绪仍会持续发酵。



# 三、关系的合法性


## 回答的问题

**这段关系有没有资格成为恋爱主线？**

## 合法性的定义

**合法性 =**
这段亲密关系，
是否在作品自身反复确认的价值体系中，
被允许
作为**主要关系**
持续存在并推进。

这里讨论的不是关系是否存在，而是是否被当作必须写完的关系



## 核心概念

### 什么是“这段亲密关系”

作者正在持续投入**最高优先级叙事资源**的那一对关系：
* 镜头时间
* 内心独白
* 情绪高潮
* 关键转折
* 明确排他性的“只有他们在场”的场景


如果这些资源在关键阶段被长期让渡给其他内容，该关系自动失去主线资格



### 什么是“作品自身的价值体系”

作品反复**奖励什么、惩罚什么、认真对待什么**。

#### 认真对待

某个行为是否被认真对待，只看一个结果：

它是否对关系产生了不可逆影响

* 告白失败 → 关系结构发生变化 → 被认真对待
* 告白失败 → 下话当没发生 → 未被认真对待


#### 奖励 / 惩罚（结构意义）
* **奖励**：该行为为后续叙事打开新的结构空间
* **惩罚**：该行为耗尽结构可能性，使关系必须转型或终结


如果一个行为：
* 既不打开新空间
* 也不迫使关系变化

那它在结构上是无效行为


```
三者的关系像是吃饭，认真对待行为本身就像是做饭，认真对待就是好吃的饭，不认真对待就是没做饭，奖励就是吃饭本身，吃饭可以无限发生（意思就是我可以分很多口吃饭），但是无论如何，饭总有被吃完的时候（观众感到无聊），但饭被吃完，你就已经饱了，这个完成的状态本身被称为惩罚，同时惩罚本身也可以成为奖励（因为你不一定吃饭吃饱了，你可以去吃其他的东西）

如果没有认真对待行为，那么出现的问题就是没做饭，吃饭本身没有意义，吃了也只是吃空气，所以会出现甜而空的番剧
```




### 什么是“被允许”

作者是否持续为这段关系：
* 让出核心叙事位置
* 承担主要冲突
* 承认其变化的后果


一旦作者开始系统性回避其后果，该关系在结构上即被判定为“不被允许”


### 什么是“主要关系”

当观众回忆这部作品时，无需解释就能回答：
 “这是在讲谁和谁。”


需要解释才能成立的关系，不是主要关




### 什么是“持续存在并推进”

* 关系不会被阶段性消耗后丢弃
* 每一次变化都会留下长期痕迹
* 不通过“当没发生过”来重置关系


## 合法性的四个并列条件

### 叙事授权

判断问题（只能是是 / 否）：

* POV 是否长期稳定围绕该关系
* 情绪高光是否持续优先分配
* 是否存在为该关系牺牲其他剧情线的情况


只要存在“可随时抽离而不影响主结构”，叙事授权判定失败。


### 冲突正当性

判断问题：
**这段关系的痛苦，是否被当作“值得承受的代价”？**

合法来源仅包括：
* 人物不可消除的性格缺陷
* 已做出的、不可撤回的选择
* 价值观层面的真实冲突

不合法表现（任一出现即失败）：
* 已解决的冲突反复出现
* 已被否定的矛盾继续占用篇幅
* 冲突结束后关系完全回到原点


### 可持续推进性

#### 核心判断问题

**这是否仍然是“同一段关系”，而不是同一组人物在另一段关系中继续出现？**
关系 = 亲密生成与推进的结构机制

本条不关心：
* 人物是否继续登场
* 情感是否仍然存在
* 故事是否仍然好看

**只关心一件事：**
这段关系在结构上，是否仍然依赖**同一套亲密生成与推进机制**存活


#### 判定原则
1. **关系不是人物属性，而是运行中的结构**
2. **后续即便继续描写这两个人，也只能视为新关系**



“阶段变化”“形式演化”“更成熟的相处方式”
**不构成同一关系的自动延续理由**。


#### 什么是作弊（判定为失败条件）
首先明确一点，我知道作弊这个词比较严重，但是我想不到一个更好更柔和的词来代替

**作弊 =** 用不属于这段关系原始结构的力量，代替关系本身完成推进。

以下任一情况出现，即判定该关系 **不具备可持续推进性**。


##### 外力接管型作弊

**定义：**
当关系自身已经无法产生推进时，
作者引入一个**更大的外部事件**强行推动关系变化。

**判断标准：**

删除该外部事件后，作者是否立刻失去推进这段恋爱的能力？
如果答案是“是”，即判定为外力接管型作弊。


**典型表现：**
* 突然出现严重疾病
* 强制分离（出国 / 转学 / 长期异地）
* 家庭或社会性巨变
* 与恋爱本身无关的终极抉择

 **重要说明：**
* 是否有铺垫
* 是否符合现实逻辑
* 是否情感上动人


**均不构成免责条件**
本条只判断这段恋爱是否还能靠自己活下去



##### 规则重置型作弊（高频误判点）

**定义：**
关系确认后，作者悄然改变了这段关系的运行规则
使其能够在不再支付原有代价的情况下继续存在。

**判断标准：**
关系确认后，人物是否仍需支付告白前的心理代价？

这句话简单理解为，马丁布伯的我与你

**一个典型作弊例子**
告白前：
女主害怕被拒绝
男主不敢越界
每一步都要压着心理重量走

告白后：
所有顾虑自动消失
冲突不再成立
情绪成本直接归零

作者心里想的是：
“他们已经在一起了嘛。”

这就是删规则

再举个例子，在马丁布伯的理论框架中，他将人的关系分成两种，一种叫我与他，一种叫我与你

极端的说，我与他是将除了自己以外的人都视作物品，而当关系进展到我与你时，我将你视为与我同级别的存在，此时规则被加重了，这是反直觉的，当你对你的父母描述你的不幸时，父母可能会担心，可能会责备，当你对陌生人说你的不幸时，陌生人大多选择安慰你



因此，以下任一出现，即判定作弊：
* 原本阻止告白的心理障碍消失
* 冲突强度在无解释情况下显著下降
* 过去的禁忌不再产生现实压力
* 曾被反复强调的风险不再被提及


即便关系仍在、篇幅仍多、互动仍甜，
只要代价被撤回，即视为规则已被重置

角色关系升级之后，角色承担的心理与伦理成本，是变少了，还是变多了？
变少了便是作弊，变多了则是升级



##### 引擎替换型作弊

**定义：**
原本推动恋爱关系的核心引擎耗尽后
作者改用**另一种叙事引擎**维持作品运转
但仍宣称“这是同一条恋爱主线”

**判断标准：**
当观众回忆这部作品时，是否会意识到它已经不在讲最初那段恋爱了？

若是，即判定为引擎替换型作弊

**典型表现：**

* 恋爱 → 群像
* 恋爱 → 社会议题
* 恋爱 → 个人成长

**关键判定句：**
只要恋爱不再是承压中心， 该关系即失去主线资个

不讨论新内容是否优秀，
只判定：
 **这段恋爱是否还能作为“推进核心”存在。**




#### 本条最终结论方式

* **全部未触发** → 关系具备可持续推进性
* **任一触发** → 原关系结构已结束
* 后续描写只可视为**新关系或其他叙事主题**



### 终局承诺

判断问题：
**作品是否承诺不撤回这段关系的意义？**

最低标准：
* 关键变化不被抹除
* 情绪线不中途失语
* 角色不会“当没发生过”

BE / 分手 / 失败不影响合法性，
撤回意义才影响




# 四、这段恋爱关系是如何变化的（压力测试）

在关系被判定为合法后，
作品必须通过以下四个测试：





### 测试一

主要冲突结束后，
关系是否进入新的存在状态？

否：关系在结构上已被用尽


### 测试二

关系变化是否不可逆？

否：空转关系




### 测试三

抽离旧推进引擎后，
关系是否仍被迫变化？

否：引擎依赖型关系



### 测试四

作品是否愿意持续为这种变化付出篇幅？

否：阶段性利用关系



## 特别说明

这一部分测试的是
 **恋爱是否还能作为主线引擎**

而不是

作品是否还能存在
人物是否还能互动
世界是否还能运转
